# Contributing to FireRed OpenStoryline

Thanks for your interest in contributing!

## Quick Start

```bash
git clone https://github.com/FireRedTeam/FireRed-OpenStoryline.git
cd FireRed-OpenStoryline
```

## Development

This project uses Python. Please ensure your code follows the existing style.

## Pull Request Process

1. Fork the repository
2. Create a branch from `main`
3. Make focused, minimal changes
4. Open a PR with clear description

## Issues

Please include:
- Clear reproduction steps
- Expected behavior
- Your environment (OS, Python version)
